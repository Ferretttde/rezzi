# Rezzi — Shared Recipe & Meal Planner PWA

A mobile-first Progressive Web App for couples to collect, organize, and plan meals together.

## Features

- **Recipe collection** — Search, filter by tags, favoriting
- **Recipe import** — Paste any URL to auto-import from AllRecipes, BBC Good Food, etc.
- **Meal planner** — Drag-and-drop weekly meal planner
- **Real-time sync** — Changes appear on your partner's phone instantly (~100ms)
- **PWA** — Install on both phones, works offline
- **Household sharing** — Invite code to link two accounts to the same kitchen

## Tech Stack

| Layer | Choice |
|---|---|
| Frontend | React 19 + Vite + TypeScript |
| Styling | TailwindCSS v4 + custom design system |
| Routing | TanStack Router v1 |
| Server state | TanStack Query v5 |
| Client state | Zustand |
| Forms | React Hook Form + Zod |
| Drag & drop | dnd-kit |
| Backend | Supabase (PostgreSQL + Auth + Storage + Realtime + Edge Functions) |
| PWA | vite-plugin-pwa (Workbox) |
| Deployment | Vercel |

## Getting Started

### 1. Clone and install

```bash
git clone <repo>
cd rezzi
npm install
```

### 2. Set up Supabase

1. Create a project at [supabase.com](https://supabase.com)
2. Run the migration: `supabase/migrations/001_initial_schema.sql` in the SQL editor
3. Enable Realtime for the `recipes`, `meal_plans`, and `recipe_tags` tables
4. Deploy the Edge Function:
   ```bash
   supabase functions deploy import-recipe
   ```

### 3. Configure environment

```bash
cp .env.example .env
# Fill in your Supabase URL and anon key
```

### 4. Run locally

```bash
npm run dev
```

The `routeTree.gen.ts` is auto-generated by TanStack Router on first `npm run dev`.

### 5. Deploy

Connect the repo to [Vercel](https://vercel.com). Add environment variables:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## Auth Flow

1. **Person A** signs up → household auto-created → gets an 8-char invite code
2. **Person B** signs up → enters invite code → linked to same household
3. Both see identical data via RLS

Find your invite code in the future Settings page (or from `households.invite_code` in Supabase).

## PWA Icons

Place the following icons in `public/icons/`:
- `icon-192.png` — 192×192
- `icon-512.png` — 512×512
- `icon-maskable-512.png` — 512×512 maskable (with safe zone)

Use [maskable.app](https://maskable.app) to generate maskable icons.
